

# **Primeira Maratona em C#: Versionamento com Git e Envio para o GitHub**

## **1. Instruções de Entrega**

Cada estudante deve enviar o link do seu repositório no GitHub contendo a pasta **maratonaexercicio01**, individualmente, em resposta a esta mensagem.

---

## **2. Procedimento de Envio para o GitHub**

Para enviar o projeto ao GitHub, utilize os comandos do **Git** por meio do **Prompt de Comando (cmd)**, realizando as etapas de *commit* e *push* conforme sua configuração local.
